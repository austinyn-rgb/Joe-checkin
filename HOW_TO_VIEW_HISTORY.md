# 📅 如何查看历史记录

## 🎯 历史记录位置

历史记录在应用底部，标题为 **"打卡历史记录"**。

---

## 📖 查看步骤

### 步骤 1：打开应用

在浏览器中打开应用链接：
```
https://joe-checkin-xxx.vercel.app
```

### 步骤 2：滚动到底部

向下滚动页面，找到 **"打卡历史记录"** 卡片。

### 步骤 3：查看记录

你会看到每天的打卡记录，按时间倒序排列（最近的在前面）。

## 💾 如何保存历史记录

由于数据存储在浏览器的 localStorage 中，寒假结束时建议保存记录。

### 方法 1：截图保存（最简单）

#### 推荐工具

- **Windows**：
  - 截图工具：Win + Shift + S
  - 或者使用微信截图（Alt + A）

- **Mac**：
  - 截图工具：Cmd + Shift + 4
  - 或者使用 QQ 截图（Cmd + Ctrl + A）

- **iPad/iPhone**：
  - 同时按下电源键 + 音量加键

#### 截图建议

**方式 1：整体截图**
- 一次性截图整个历史记录区域
- 适合记录少的场景

**方式 2：分批截图**
- 每周或每月截图一次
- 更清晰，便于分类整理

**方式 3：每日截图**
- 每天打卡后截图保存
- 最详细，但工作量较大

---

### 方法 2：复制文本

#### 步骤

1. 在浏览器中打开应用
2. 选中想要保存的记录文本
3. 右键 → 复制（或 Ctrl + C）
4. 粘贴到文档中（Word、记事本等）

#### 适合场景

- 需要文字记录
- 需要编辑或整理
- 需要长期保存

---

### 方法 3：打印页面

#### 步骤

1. 打开应用
2. 按 Ctrl + P（Windows）或 Cmd + P（Mac）
3. 选择打印到 PDF
4. 保存 PDF 文件

#### 适合场景

- 需要完整保存
- 需要打印成纸质版
- 需要长期归档

---

### 方法 4：导出数据（需要浏览器开发者工具）

⚠️ **高级用户使用**

#### 步骤

1. 打开应用
2. 按 F12 打开开发者工具
3. 切换到 **"Console"**（控制台）标签
4. 输入以下命令：

```javascript
// 1. 获取所有历史记录
const history = JSON.parse(localStorage.getItem('checkInHistory'));

// 2. 格式化输出
console.table(history.map(h => ({
  日期: h.date,
  完成状态: h.completed ? '已完成' : '未完成',
  积分: h.completed ? 500 : 0,
  详细内容: h.completed ? 
    Object.entries(h.activities)
      .filter(([_, a]) => a.completed)
      .map(([k, a]) => `${k}: ${(a as any).content || '未填写'}`)
      .join(', ') 
    : '无'
})));

// 3. 复制数据
copy(JSON.stringify(history, null, 2));
```

5. 复制输出的数据
6. 粘贴到文本编辑器保存

## 📈 数据统计分析

### 方法 1：手动统计

#### 连续打卡天数

在应用顶部可以直接看到：
- **"连续打卡天数"** 卡片
- 显示总共连续打卡了多少天

#### 累计总积分

在应用顶部可以直接看到：
- **"累计积分"** 卡片
- 显示总共获得了多少积分

#### 完成次数统计

在历史记录中：
- 每条记录显示日期
- 数一数有多少条记录
- 就是完成了多少天

### 方法 2：使用 Excel 整理

#### 步骤

#### 可做的分析

- 每周完成情况

- 阅读的书单统计
- 锻炼内容统计

| 频率 | 方法 | 说明 |

## ⚠️ 注意事项

### 数据丢失风险

1. **清除浏览器数据**
   - ❌ 不要清除浏览器缓存
   - ❌ 不要删除 Cookies
   - ❌ 不要清除网站数据

2. **更换设备**
   - ❌ 数据不会自动同步
   - ✅ 需要提前备份

3. **浏览器升级**
   - ⚠️ 通常不会丢失数据
   - ✅ 建议提前备份
