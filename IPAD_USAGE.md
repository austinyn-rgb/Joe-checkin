# 📱 iPad 使用说明 - 重要必读

## 核心问题解答

### 问题：在自己的电脑上部署，小朋友在 iPad 上打开有问题吗？

### 答案：完全没问题！✅

**但是有一个重要的前提条件**：
- ✅ 您必须使用 **Vercel 部署到云端**，而不是只在本地运行
- ✅ 部署后获得的**公网链接**可以在任何设备上访问（电脑、iPad、手机）
- ❌ 如果只是运行 `localhost:5000`，iPad 上无法访问

---

## 🚀 正确的部署方式

### ❌ 错误方式：只在本地运行

```bash
# 这样只是在您的电脑上运行
coze dev
# 或
vercel dev

# 地址是 http://localhost:5000
# iPad 无法访问！
```

### ✅ 正确方式：部署到云端

```bash
# 在您的电脑上执行
vercel --prod

# 部署成功后获得公网链接
# 例如：https://Joe-checkin.vercel.app
# 这个链接可以在任何设备访问！
```

---

## 📱 在 iPad 上的使用方式

### 步骤 1：获取部署链接

1. 在您的电脑上部署到 Vercel
2. 复制获得的链接，如：`https://Joe-checkin.vercel.app`

### 步骤 2：在 iPad 上打开

#### 方法一：通过飞书打开（推荐）
1. 您在飞书中发送链接给小朋友
2. 小朋友在 iPad 上的飞书里点击链接
3. 自动在浏览器中打开

#### 方法二：直接用 Safari 打开
1. 打开 iPad 上的 Safari 浏览器
2. 在地址栏输入链接
3. 按回车键

#### 方法三：保存到主屏幕
1. 在 Safari 中打开链接
2. 点击底部的"分享"按钮（方框+向上箭头）
3. 选择"添加到主屏幕"
4. 给应用起个名字，如"杨子祐打卡"
5. 点击"添加"
6. 现在 iPad 主屏幕上就有图标了，像普通 App 一样打开！

### 步骤 3：开始打卡

1. 打开应用
2. 完成当天 5 项活动
3. 勾选对应的复选框
4. 点击"完成全部任务，领取积分！"
5. 数据自动保存在 iPad 上

## 💾 数据存储说明（非常重要！）

### 数据存在哪里？

**数据存储在设备浏览器的 localStorage 中**：
- ✅ 在 iPad 上打卡 → 数据保存在 **iPad 的 Safari 浏览器** 中
- ✅ 在电脑上打卡 → 数据保存在 **电脑的浏览器** 中
- ❌ 数据不会自动在不同设备间同步

### 这意味着什么？

#### 场景 1：只在 iPad 上使用 ✅（推荐）
- 小朋友每天都在 iPad 上打卡
- 数据正常保存在 iPad 上
- 积分和打卡记录会一直累积
- **只要不清除浏览器数据，数据不会丢失**

#### 场景 2：有时在 iPad，有时在电脑 ⚠️
- 周一到周五在 iPad 上打卡
- 周末在电脑上打卡
- **两个设备的数据是独立的**
- iPad 上看不到电脑上的打卡记录
- 电脑上也看不到 iPad 上的打卡记录
- **建议固定使用一台设备！**

#### 场景 3：换了新 iPad ⚠️
- 旧 iPad 上的数据不会自动迁移到新 iPad
- 需要重新开始打卡
- **建议更换前截图保存记录**

## ⚠️ 数据丢失风险

### 什么情况下数据会丢失？

1. **清除浏览器缓存/数据**
   - Safari 设置 → 清除历史记录与网站数据
   - 数据会被清空！

2. **卸载 Safari**
   - iPad 上通常无法卸载 Safari
   - 但如果重置 iPad，数据会丢失

3. **使用无痕模式**
   - 无痕模式关闭后数据会被清除
   - **不要用无痕模式打卡！**

4. **使用隐私保护浏览器**
   - 某些隐私浏览器会自动清除数据
   - **建议使用系统自带的 Safari**

5. **换设备**
   - iPad 上的数据无法迁移到新设备
   - 需要重新开始

### 如何防止数据丢失？

#### ✅ 推荐做法

1. **固定使用一台设备**
   - 建议只在一台 iPad 上打卡
   - 不要频繁切换设备

2. **不要清除浏览器数据**
   - 告诉小朋友不要点击"清除历史记录"
   - 告诉小朋友不要清除 Safari 缓存

3. **定期截图备份**（非常重要！）
   - **每周截图一次**积分页面
   - **每月截图一次**打卡历史
   - 保存到相册或电脑

4. **添加到主屏幕**
   - 方便快速访问
   - 减少误操作清除数据的风险

#### 📋 截图备份清单

建议家长按照以下频率截图：

| 频率 | 截图内容 | 保存位置 |
|------|---------|---------|
| 每周 | 积分页面（总积分、连续打卡天数） | iPad 相册 |
| 每周 | 打卡历史记录 | iPad 相册 |
| 每月 | 完整页面截图 | 电脑备份 |

## 🆘 常见问题

### Q1：为什么我在电脑上能看到链接，但 iPad 打不开？

**可能原因**：
- 链接不是公网地址（如 `localhost:5000`）
- 网络连接问题

**解决方法**：
- 确保使用 Vercel 部署获得的公网链接
- 检查 iPad 网络连接

### Q2：小朋友换了新 iPad，数据能迁移吗？

**答案**：不能自动迁移

**解决方法**：
- 换设备前截图保存所有记录
- 新设备重新开始打卡
- 记录之前的积分（手动累计）

### Q3：iPad 上不小心清除了浏览器数据，能恢复吗？

**答案**：无法恢复

**预防方法**：
- 定期截图备份
- 告诉小朋友不要清除 Safari 数据

### Q4：可以在 iPhone 上和 iPad 共用吗？

**答案**：不推荐

**原因**：
- 两个设备数据独立
- 不会自动同步
- 容易混淆记录

**建议**：
- 固定在一台设备上使用

### Q5：飞书打不开链接怎么办？

**解决方法**：
- 方法 1：直接在 Safari 中输入链接
- 方法 2：复制链接，在 Safari 中粘贴
- 方法 3：检查网络连接
