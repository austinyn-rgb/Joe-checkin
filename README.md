## 数据存储说明 💾

### 当前存储方式
**存储位置：浏览器的 localStorage（本地存储）**

- 数据存储在小朋友使用的**浏览器/设备上**
- 每次打卡后，数据会自动保存到该设备
- 刷新页面或关闭浏览器后，数据**不会丢失**

### 如何保证数据不被丢失？

#### ✅ 优点
- 数据自动保存，无需手动操作
- 关闭浏览器后数据仍保留
- 访问速度快，无需联网

#### ⚠️ 注意事项（重要！）

1. **设备固定性**
   - 建议小朋友在**同一台设备**上使用（如家里的iPad或手机）
   - 如果换设备打卡，数据不会同步

2. **不要清除浏览器数据**
   - 告诉小朋友不要清除浏览器缓存和数据
   - 否则打卡记录会被清空

3. **定期截图备份**（推荐）
   - 建议家长每周或每月截图保存积分和打卡记录
   - 以防万一设备更换或数据丢失

### 如何查看数据？

- 每日的打卡状态

## 每日打卡流程 📝

### 打卡步骤

1. 打开应用
2. 查看当天的5项活动：
   - 🎨 寒假生活（45分钟）
   - 📚 中文阅读（45分钟）
   - 🔢 数学练习（45分钟）
   - 🌍 英文阅读（45分钟）
   - 🏃 体育锻炼（45分钟）
3. 完成一项活动后：
   - 在输入框中填写具体内容（如书名、页数等）
   - 勾选对应的复选框
4. 对每项活动重复步骤 3
5. 完成后点击"🎯 提交打卡"按钮
6. 系统自动记录：连续打卡天数 + 积分

## 积分规则 🏆

### 基本规则

- 每完成一项活动：+100 积分
- 完成几项获得几项的积分（如 3 项 = 300 分）
- 每天最多可获得：500 积分（完成全部 5 项）
- 系统自动计算：累计总积分 + 连续打卡天数

### 积分示例

| 完成项数 | 获得积分 |
|---------|---------|
| 1 项 | 100 积分 |
| 2 项 | 200 积分 |
| 3 项 | 300 积分 |
| 4 项 | 400 积分 |
| 5 项 | 500 积分 |

- **累计积分**：显示总共获得了多少积分

## 数据导出功能 📦

### 三种导出格式

寒假结束时，可以导出所有打卡数据：

1. **CSV 格式**（详细记录）
   - 用 Excel 打开
   - 包含每日详细信息和统计摘要
   - 适合数据分析和报告

2. **JSON 格式**（完整备份）
   - 完整的数据备份
   - 包含所有详细信息
   - 适合技术处理

3. **TXT 格式**（汇总报告）
   - 格式化的统计报告
   - 便于阅读和打印
   - 适合分享给家长或老师

### 如何导出

1. 滚动到"数据导出"卡片
2. 点击对应的导出按钮
3. 文件会自动下载

**详细教程**：[数据导出使用指南](HOW_TO_EXPORT_DATA.md)

## 常见问题 FAQ

### 关于打卡

**Q: 必须完成全部 5 项才能打卡吗？**
A: 不需要。现在支持部分打卡，至少完成 1 项即可打卡。完成几项获得几项的积分。

**Q: 如何修改今天的内容？**
A: 当前版本不支持修改。建议打卡前仔细检查，如需修改功能可以联系开发者。

**Q: 可以补卡吗？**
A: 当前版本不支持补卡。建议每天按时打卡。

---

### 关于数据

**Q: 数据会同步到云端吗？**
A: 不会。数据保存在设备浏览器中，不跨设备同步。建议定期导出数据备份。

**Q: 小朋友换了设备怎么办？**
A: 建议在更换前导出数据（JSON 格式）。新设备需要重新开始打卡。

**Q: 数据会丢失吗？**
A: 如果清除浏览器数据或更换设备，数据会丢失。请定期导出数据备份。

---

### 关于部署

**Q: 部署需要花钱吗？**
A: 不需要。Vercel 免费版完全够用，包含 100GB/月 带宽和无限项目数量。

**Q: 可以多人使用同一个应用吗？**
A: 可以，但每个人的数据是独立的。每台设备的打卡记录互不干扰。

**Q: 如何修改应用名称？**
A: 修改 `src/app/page.tsx` 中的标题文字，提交到 GitHub，Vercel 会自动更新。

**Q: 如何修改积分规则？**
A: 修改代码中的积分计算逻辑，提交到 GitHub，Vercel 会自动更新。修改只影响未来的打卡。

---

### 关于使用

**Q: 在 iPad 上如何使用？**
A: 在 Safari 中打开网页，点击分享按钮 → 添加到主屏幕，像 App 一样使用。

**Q: 如何分享到飞书？**
A: 复制网页链接，在飞书中粘贴发送。也可以使用网页内的"分享到飞书"按钮。

**Q: 可以在微信中使用吗？**
A: 可以。复制链接到微信中发送即可。
