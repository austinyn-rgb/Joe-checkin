# ⚡ 快速开始指南（5 分钟版本）

## 🚀 最快部署流程（推荐）

如果你有技术基础，可以使用这个快速版本。

---

## 第一步：从沙箱下载代码（2 分钟）

1. 在沙箱中找到项目根目录
2. 选中所有文件（`src/`, `package.json`, `.coze`, `tsconfig.json` 等）
3. 右键 → 压缩为 ZIP 文件
4. 下载到电脑

---

## 第二步：上传到 GitHub（2 分钟）

### 2.1 创建仓库

1. 访问：https://github.com/new
2. **Repository name**: `winter-checkin-app`
3. 选择 **Public**（公开）
4. 点击 **"Create repository"**

### 2.2 上传代码（方式 A：网页上传）

1. 点击 **"uploading an existing file"** 链接
2. 拖拽刚才的 ZIP 文件到页面
3. 解压文件（如果需要）
4. 输入提交信息：`Initial commit`
5. 点击 **"Commit changes"**

### 2.3 上传代码（方式 B：使用 GitHub Desktop）

1. 下载安装：https://desktop.github.com/
2. 登录 GitHub
3. File → Add Local Repository → 选择解压后的文件夹
4. Commit → Publish repository

---

## 第三步：部署到 Vercel（1 分钟）

1. 访问：https://vercel.com/
2. Login → Continue with GitHub
3. Add New → Project
4. 找到 `winter-checkin-app` → Import
5. 点击 **"Deploy"**
6. 等待 1-3 分钟
7. 复制域名：`https://winter-checkin-app.vercel.app`

---

## 第四步：测试和分享（1 分钟）

1. 在浏览器打开域名
2. 测试打卡功能（填写内容 → 勾选 → 提交）
3. 复制链接，发送到飞书

---

## ✅ 完成！

现在你可以在任何设备上使用应用了！

---

## 📱 在 iPad 上使用

1. 在 Safari 打开网页
2. 点击分享按钮 → 添加到主屏幕
3. 以后像 App 一样点击打开

---
