# 📦 数据导出功能使用指南

## 🎯 功能概述

数据导出功能允许你在寒假结束时导出所有打卡数据，方便汇总、存档和分享。支持三种导出格式：

1. **CSV 格式** - 详细打卡记录，可以用 Excel 打开
2. **JSON 格式** - 完整数据备份，适合技术处理
3. **TXT 格式** - 统计汇总报告，便于阅读和打印

---

## 📍 导出功能位置

导出功能位于应用底部，标题为 **"数据导出"** 卡片。

### 如何找到导出区域

1. 打开应用
2. 向下滚动页面
3. 找到 **"数据导出"** 卡片（蓝色边框）
4. 你会看到三个导出按钮

---

## 🚀 三种导出方式

### 1. 导出详细记录（CSV 格式）

**图标**：📊 绿色表格图标

**适用场景**：
- 查看详细的每日打卡记录
- 用 Excel 进行进一步分析
- 生成表格报告

**导出内容**：
```
日期, 是否打卡, 寒假生活, 寒假生活内容, 中文阅读, 中文阅读内容, 
数学练习, 数学练习内容, 英文阅读, 英文阅读内容, 体育锻炼, 体育锻炼内容, 
完成项数, 获得积分
2025-01-15, 是, ✓, 画画和做手工, ✓, 《西游记》第10-20页, 
✓, 口算题卡第5页, ✓, 绘本第5-10页, ✓, 跑步30分钟, 5, 500
...

=== 统计摘要 ===
总打卡天数, 15
连续打卡天数, 12
累计积分, 7200
平均每日积分, 480
```

**如何打开**：
1. 用 Microsoft Excel 打开
2. 用 Google Sheets 打开
3. 用 Numbers (Mac) 打开
4. 任意文本编辑器查看

## 📝 使用步骤

### 基本导出步骤

1. **滚动到底部**
   - 打开应用
   - 向下滚动到"数据导出"卡片

2. **选择导出方式**
   - 根据需要点击三个按钮之一
   - CSV：导出详细记录（推荐用于表格分析）
   - JSON：导出完整数据（推荐用于备份）
   - TXT：导出汇总报告（推荐用于阅读和打印）

3. **下载文件**
   - 点击按钮后，文件会自动下载
   - 文件名格式：`杨子祐寒假打卡[类型]_日期.扩展名`
   - 例如：`杨子祐寒假打卡记录_2025-02-14.csv`

4. **打开文件**
   - 根据文件类型选择相应的程序打开

## 🔧 常见问题

### Q1: 为什么导出时提示"暂无打卡数据可导出"？

**A**：这是因为还没有任何打卡记录。需要先完成至少一次打卡后才能导出数据。

### Q2: CSV 文件用 Excel 打开后乱码怎么办？

**A**：
1. 在 Excel 中打开 CSV 文件
2. 点击"数据" → "从文本/CSV"
3. 选择文件编码为 UTF-8
4. 点击"加载"

或者使用 Google Sheets 打开，会自动识别编码。

### Q3: 可以修改导出的文件内容吗？

**A**：
- CSV 和 TXT 文件可以用任意文本编辑器修改
- JSON 文件也可以修改，但要确保格式正确
- 修改后的文件不会影响应用中的数据

### Q4: 导出的数据可以重新导入吗？

**A**：
- 当前版本不支持重新导入
- 建议定期导出 JSON 格式作为备份
- 如需导入功能，可以后续开发

### Q5: 多次导出会覆盖之前的文件吗？

**A**：
- 不会，每次导出都会生成新的文件
- 文件名包含日期，方便区分
- 建议定期清理旧的导出文件

### Q6: 可以导出部分时间范围的数据吗？

**A**：
- 当前版本导出全部历史数据
- 如需按时间范围导出，可以使用 Excel 筛选 CSV 文件
- 或者等待后续更新此功能

## 🎯 实际使用场景

### 场景 1：寒假结束，提交给家长

**步骤**：
1. 滚动到"数据导出"卡片
2. 点击"导出汇总报告"（紫色按钮）
3. 下载 TXT 文件
4. 打开文件查看
5. 打印或发送给家长

**结果**：得到一份完整的寒假打卡汇总报告

---

### 场景 2：分析阅读完成情况

**步骤**：
1. 点击"导出详细记录"（绿色按钮）
2. 下载 CSV 文件
3. 用 Excel 打开
4. 筛选"中文阅读"列
5. 统计完成次数和内容

**结果**：可以看到中文阅读的详细完成情况

---

### 场景 3：完整备份数据

**步骤**：
1. 点击"导出完整数据"（蓝色按钮）
2. 下载 JSON 文件
3. 保存到云盘或电脑
4. 定期重复此步骤

**结果**：拥有完整的数据备份，防止数据丢失
